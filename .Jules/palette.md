## 2025-05-15 - Consistent Search Clearing Pattern
**Learning:** In applications with multiple independent search context (e.g., global sidebar search, local marketplace search, and mobile-specific search), users expect a consistent "Clear" button interaction to reset filters quickly.
**Action:** Always provide a clear button using the standard 'X' icon (Lucide) for all search inputs that have active text, ensuring they are accessible with ARIA labels and focus rings.
