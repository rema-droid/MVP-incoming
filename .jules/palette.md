## 2026-04-10 - [Clear Button in Search Inputs]
**Learning:** All search inputs (Sidebar, Marketplace, Mobile) should provide a conditional 'Clear' button (using an 'X' icon, type="button", and aria-label="Clear search") when text is entered to improve usability on touch devices. To prevent search text from overlapping absolute-positioned 'Clear' buttons, use pr-9 or pr-10 padding on search input elements.
**Action:** Always include a clear button for search inputs and ensure sufficient right padding to avoid overlap.
