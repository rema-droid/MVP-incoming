## 2025-04-14 - Standardized Clear Search Pattern
**Learning:** Consistent search UX across desktop, mobile, and marketplace views improves user confidence. When adding absolute-positioned clear buttons, input padding-right must be adjusted (pr-9/pr-10) to prevent text overlap, which is a common accessibility/visual regression.
**Action:** Always check all search implementations when adding global UX improvements to ensure consistent behavior across viewports and specialized views.
